O que é HTML?

HTML é a linguagem de marcação utilizada para estruturar o conteúdo da web.

O que é CSS?

CSS é a linguagem usada para estilizar o HTML e deixar as paginas mais bonitas.

Exemplos de Códigos

Exemplo de um titulo em HTML:

<h1>Olá Mundo<h1>

Exemplo de estilização em CSS:
